# Wedding-Invitation

موقع دعوة زفاف تفاعلي فاخر باسم **غانم ❤️ أمنية**، بتصميم أبيض وذهبي بأسلوب Glassmorphism، ومجهز للرفع مباشرة على GitHub Pages.

## المزايا

- HTML5 + CSS3 + JavaScript بدون أي إطار عمل.
- ظرف ثلاثي الأبعاد مبني باستخدام Three.js، يفتح عند الضغط أو باستخدام Enter / Space.
- حركة خروج بطاقة الدعوة من الظرف بتأثير سينمائي باستخدام GSAP.
- تصميم متجاوب مع الجوال والأجهزة اللوحية والشاشات الكبيرة.
- عداد تنازلي حتى موعد الزفاف.
- مشغل لأغنية **هو حبيبي — أصالة** مع إمكانية استبدال ملف الصوت من داخل الموقع أو بتبديل الملف داخل المشروع.
- مساحة أنيقة لإضافة صورة غانم وأمنية، مع إمكانية اختيار صورة من الجهاز.
- ورود / بتلات متساقطة ولمعات ذهبية متحركة.
- ألعاب نارية تعمل عند الوصول إلى نهاية الصفحة، وكذلك عند انتهاء العداد.
- دعم `prefers-reduced-motion` لتقليل الحركة للمستخدمين الذين يفضلون ذلك.
- لا توجد إعدادات بناء أو تثبيت حزم؛ الملفات تعمل مباشرة في المتصفح.

## تشغيل المشروع محلياً

يمكن فتح `index.html` مباشرة في المتصفح. ولتجربة أفضل مع جميع المتصفحات، استخدم خادماً محلياً بسيطاً من داخل مجلد المشروع:

```bash
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## إضافة أو استبدال الأغنية

### الطريقة الأولى: ملف المشروع

1. أنشئ مجلدًا باسم `assets` بجوار `index.html`.
2. ضع ملف MP3 داخله.
3. سمّه:

```text
assets/asala-howa-habibi.mp3
```

المسار مستخدم بالفعل في `index.html`، لذلك لا يلزم تعديل الكود.

### الطريقة الثانية: من داخل الموقع

اضغط زر **اختيار أغنية بديلة** في بطاقة الدعوة واختر ملف MP3 من جهازك. هذا الاستبدال يعمل للجلسة الحالية في المتصفح فقط، ولا يرفع الملف إلى GitHub.

> ملاحظة: المتصفحات تمنع التشغيل التلقائي للصوت غالباً، لذلك يبدأ الصوت بعد ضغط المستخدم على زر التشغيل.

## إضافة الصورة

الصورة الحالية مضمّنة داخل `index.html` حتى تظهر حتى لو حدثت مشكلة في مسار مجلد `assets`. ويمكنك أيضاً الاحتفاظ بنسخة الصورة داخل المسار التالي:

```text
assets/ghanim-omnia-photo.jpg
```

لتغيير الصورة مستقبلاً، استبدل قيمة `src` الخاصة بالعنصر `couplePhoto` أو استخدم زر **اختيار صورة من الجهاز** داخل الدعوة لمعاينة صورة مختلفة في الجلسة الحالية فقط.

يفضل استخدام صورة طولية أو مربعة عالية الجودة بصيغة JPG أو PNG أو WebP.

## تعديل التاريخ

التاريخ الحالي في المشروع مضبوط على:

```text
الجمعة 14/8 — 14 أغسطس 2026
```

تم اختيار سنة 2026 لأن 14 أغسطس 2026 يوافق يوم الجمعة. إذا كان المقصود سنة أخرى، غيّر السطر التالي في `script.js` فقط:

```javascript
const WEDDING_DATE = new Date('2026-08-14T00:00:00+03:00').getTime();
```

ويمكن تعديل النص الظاهر في `index.html` داخل قسم تفاصيل الحدث.

## تعديل البيانات الظاهرة

- الأسماء: عدّل النصوص `غانم` و`أمنية` في `index.html`.
- المكان: عدّل `قاعة البانسيه` في `index.html`.
- عنوان الصفحة والوصف: عدّل عناصر `title` و`meta description` في أعلى `index.html`.
- مدة العداد: عدّل `WEDDING_DATE` داخل `script.js`.

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub.
2. ارفع الملفات والمجلد `assets` إلى المستودع.
3. من إعدادات المستودع اذهب إلى:
   `Settings → Pages`
4. من **Build and deployment** اختر:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. اضغط **Save** وانتظر إنشاء الرابط.

بعد النشر، تأكد من رفع ملفي `assets/asala-howa-habibi.mp3` و`assets/ghanim-omnia-photo.jpg` إذا أردت ظهور الأغنية والصورة من رابط GitHub Pages.
